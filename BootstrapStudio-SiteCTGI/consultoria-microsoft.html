  • Cadastro de novos usuários com suas permissões, conta de e-mail, acesso a portais, impressoras e outros recursos da rede;
  • Cancelamento de usuários;
  • Mudança de nível organizacional;
  • Situações especiais como férias, afastamentos, etc.

    A administração poderá ser feita por pessoal não familiarizado com as ferramentas da Microsoft.

    O conceito dos serviços de terminais já data desde os primeiros Mainframes, onde a "sessão" de trabalho do usuário reside no servidor.
    A implantação de servidores Windows com Terminal Services permite ganhos como:

  • Desktop padrão Windows XP em equipamentos antigos;
  • Padronização do ambiente de trabalho;
  • Acesso ao desktop de qualquer ponto de rede ou via Internet;
  • O armazenamento dos arquivos de trabalho no servidor permite que se dispense o backup dos desktop;
  • Administração centralizada, com total controle sobre os softwares instalados, atualizações aplicadas, controle de uso, etc;
  • Otimização do uso de licensas.
